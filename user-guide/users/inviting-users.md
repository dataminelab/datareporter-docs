---
title: Inviting Users to Use Redash
section: user-guide
category: users
---

Users can be invited by Admins only - to invite a new user go to `Settings`>`Users` and click on `New User`.

![](/docs/images/gitbook/invite-user.png)

Next, you'll fill in their name and email. They'll get an invite via email and be required to set up a DataReporter account.

![](/docs/images/gitbook/invite-user-2.png)

To add a user to an existing group, go to `Setting`>`Groups`, select the group and add users by typing their name:

![](/docs/images/gitbook/view-only-groups.png)
